Projects

Through academic projects, I developed a web application for IT asset management, analyzed employee attrition using Tableau and Power BI, designed a database for an airline reservation system, and built a predictive tool for mental health prediction. These projects equipped me with practical skills in data analysis, visualization, database design, and predictive modeling, laying a strong foundation for my transition into the field of data analytics.

Network Asset Management Web Application

In collaboration with my fellow students, we developed a comprehensive web application for managing IT assets. Implemented analytics, asset inventory, and a product solution finder. Utilized Figma, Lucid Chart, and dBdiagram.io for planning, VSC as code editor, and MERN (MongoDB, ExpressJS, React, NodeJS) stack for database, frontend, and backend development. Implemented Plotly for analytics dashboard, Puppeteer for scraping, and Python-Paramiko Lib for remote access to network devices.
Visit my NetHive at GitHub

Employee Attrition Analytics with Tableau and Power BI

As part of an academic project, I analyzed employee attrition data using Tableau and Power BI. Utilizing a dataset comprising 1470 employee profiles, I employed data visualization techniques to identify factors influencing employee resignation and forecast attrition rates. This project equipped me with valuable skills in data analysis and visualization, laying the foundation for my journey into the field of data analytics.

Airline Reservation System Database Design

In a database design course, I led a project focused on designing a database for an airline reservation system. Collaborating with team members, I designed the database schema using the star model, incorporating entities such as booking, passenger, flight, date, time, and aircraft. By generating and analyzing 1000 records using MongoDB, we gained insights to enhance airline service planning and management.

Mental Health Prediction Analysis

In a predictive analytics course, I undertook a project aimed at developing a predictive tool for mental health practitioners. Leveraging a dataset of 2000 patient survey records, I performed data cleaning, scaling, and modeling techniques to predict future mental health crises. This project not only honed my skills in data preprocessing and modeling but also deepened my understanding of the potential applications of data analytics in healthcare.
